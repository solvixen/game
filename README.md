# 游戏数据实时监控系统

基于 Vue 3 开发的游戏数据实时监控系统，提供服务器状态监控、玩家数据分析、实时告警等功能。

## 技术栈

- **前端框架**: Vue 3 + Composition API
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **UI组件**: Element Plus
- **数据可视化**: ECharts 5
- **实时通信**: WebSocket (Socket.io-client)
- **构建工具**: Vite

## 功能特性

### 1. 实时监控
- 服务器状态实时监控（CPU、内存、延迟）
- 在线玩家数量实时统计
- 游戏收入实时追踪
- 多服务器集群状态全景展示

### 2. 数据可视化
- 折线图展示趋势数据
- 仪表盘展示实时负载
- 热力图展示玩家分布
- 图表联动与数据缩放

### 3. 智能告警
- 阈值告警（CPU > 80%）
- 趋势告警（玩家数骤降）
- 异常检测（延迟突增）
- 多方式通知（邮件、短信）

### 4. 数据分析
- 玩家留存分析
- 付费行为分析
- 服务器性能分析
- 自定义报表导出

### 5. 系统管理
- 多角色权限控制
- 监控规则配置
- 告警规则管理
- 系统日志查看
