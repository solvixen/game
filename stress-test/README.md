# 压力测试套件

## 前提条件

后端必须先启动：

```bash
cd backend
node src/app.js
```

## 运行方式

```bash
# 运行全部5项（约需 3-5 分钟）
node stress-test/run-all.js

# 单独运行某一项
node stress-test/run-all.js 1    # 测试一
node stress-test/run-all.js 2    # 测试二
node stress-test/run-all.js 3    # 测试三
node stress-test/run-all.js 4    # 测试四
node stress-test/run-all.js 5    # 测试五
```

## 各项说明

| # | 文件 | 测什么 | 耗时 |
|---|------|--------|------|
| 1 | test-1-ws-concurrent.js | 100个WS并发连接 | ~10s |
| 2 | test-2-ws-throughput.js | 10客户端×30s消息吞吐 | ~30s |
| 3 | test-3-http-stress.js | 5个HTTP接口各压15s | ~75s |
| 4 | test-4-db-write.js | 800条DB写入+混合读写 | ~30s |
| 5 | test-5-reconnect.js | 断线重连+心跳验证 | ~30s |
